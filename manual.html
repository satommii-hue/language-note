使えなくなる可能性はほぼないと思います。VercelもGitHubも業界標準で、急に終わるものではないです。

ブラウザの互換性について

このアプリはHTMLとJavaScriptだけで動いていて、特別な機能を使っていないので、古いブラウザでも動く設計です。気になるとすれば一点だけ——音声読み上げ機能はブラウザの Speech Synthesis API を使っていて、古いブラウザだと「このブラウザは音声未対応です」と表示されます。ただそれだけで、フレーズの追加・編集・タグ管理・エクスポートなどは5年前のスマホでも普通に動きます。

データが消えるリスクについて

アプリ側の問題でデータが消えることはほぼないですが、ブラウザのキャッシュ削除・機種変更・アプリの再インストールでは消えます。定期的にエクスポートしておくのをおすすめします。

02

はじめ方

URL language-note.vercel.app

上のURLをブラウザで開くだけです。インストールは不要。

iPhone でホーム画面に追加する場合

Safariで開く → 下部の共有ボタン（四角から矢印が出てるアイコン）→「ホーム画面に追加」→ アプリっぽく使えます

03

翻訳機能のセットアップ

自動翻訳を使うには、DeepLという翻訳サービスの無料アカウントが必要です。月50万文字まで無料で使えます（日常使いで上限に達することはほぼないです）。

01
DeepLのアカウントを作る
deepl.com/ja/pro-api にアクセス。「無料でAPIキーを取得」→ メールアドレスで登録。クレジットカードの登録が必要ですが、月50万文字までは無料で、日常使いの範囲で請求されることはほぼありません。
02
APIキーをコピーする
ログイン後、「アカウント」→「APIキー」のページに移動。表示されているキー（英数字の長い文字列）をコピーします。
03
アプリに入力する
アプリを開く → 下部タブの「SET」→「DeepL APIキー」→「設定する」→ コピーしたキーを貼り付けて保存。
NOTE

APIキーはあなたのブラウザにのみ保存されます。他の人には見えません。デバイスを変えるときは再入力が必要です。

04

データのバックアップと共有

フレーズデータはブラウザ内に保存されます。PCとスマホで同じデータを使いたいときや、バックアップしたいときはエクスポート/インポートを使います。

書き出す
エクスポート → Googleドライブに保存
下部タブの「SET」→「JSONファイルを書き出す」→ 保存先にGoogleドライブを選ぶ（Chromeなら保存先ダイアログが開きます）。SafariはDownloadsに保存されるので、そこからGoogleドライブに移動。
読み込む
Googleドライブから → インポート
GoogleドライブからJSONファイルをダウンロード → アプリの「JSONファイルを選択」でそのファイルを選ぶ。既存のデータに追加（重複はスキップ）されます。
NOTE

GoogleドライブのほかにiCloudドライブ・Dropbox・OneDriveなど、ファイルを保存できる場所ならどこでも使えます。

Googleドライブ 無料容量 15 GB
語学ノートのJSONファイルは1フレーズあたり約1KB。1000フレーズ登録しても1MB以下。容量はほぼ気にしなくて大丈夫です。
05

主な使い方

追加
フレーズを追加する
「ADD」タブ → 日本語を入力 →「✨ 自動翻訳」→ 翻訳結果を確認して「追加する」。タグをつけておくとカテゴリで絞り込めます。
編集
フレーズを編集・削除する
「LIST」でカードを長押し（0.5秒）→ 編集モーダルが開きます。ここから内容の変更・削除ができます。
表示
翻訳を見る・音声を聞く
カードをタップすると各言語の翻訳が展開します。言語ごとの ▶ ボタンで発音を再生。
設定
言語を増やす・整理する
下部タブの「SET」→「言語の管理」から言語の表示/非表示を切り替えられます。非表示にしてもデータは残ります。
archive
フレーズをアーカイブする
カードを長押し → 編集モーダルの「ARCHIVE」ボタン → 一覧から消えてARCHIVEタブへ移動します。削除ではないので、あとから復元できます。
