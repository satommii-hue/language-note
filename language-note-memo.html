作業の流れ

STEP 01
「こういうものが欲しい」をClaudeに話す
多言語のフレーズを記録できるノートアプリを作りたい、と伝えた。仕様書とか書いてない。会話しながら形にしていった。
Claude Code アイデア出し
STEP 02
HTMLファイルとして出力してもらう
最初のバージョンをHTMLで書いてもらった。1ファイルで完結する形。サーバー不要で手元で開くだけで動く。
language-note.html Claude Code
STEP 03
DeepL APIキーを取得する
翻訳機能を使うために必要。DeepLのサイトでアカウントを作り、APIキーを発行した。キーはアプリの設定画面から入力する仕組み。
DeepL API アカウント作成
STEP 04
フィードバックを繰り返す
「フォントをサンセリフに」「この機能追加して」「動かなくなった」など、気になったことをその都度Claudeに伝えて直してもらった。チャットが長くなってきたら新しい会話に切り替えた（トークン対策）。
トラブル対応 反復改善
STEP 05
GitHubにアップロードする
ファイルを保管・管理するためにGitHubへ。バージョン管理ができる（＝前の状態に戻せる）のが便利。
GitHub バージョン管理
STEP 06
Vercel（またはNetlify）で公開する
GitHubと連携するだけでURLが発行される。スマホからもアクセスできるようになった。
Vercel 公開
STEP 07
Googleドライブでバックアップ管理
アプリのデータ（フレーズ一覧）をJSONでエクスポートして、Googleドライブに保管。端末が変わっても引き継げる。

Claude / Claude Code
AI アシスタント — Anthropic
▼

会話でコードを書いてもらえるAI。「こういうものが作りたい」と話すだけで、HTMLやJavaScriptを書いてくれる。Claude Codeはターミナル（黒い画面）から使うバージョンで、ファイルを直接作ったり編集したりできる。

このアプリでの役割：アプリ全体のコードを書いてもらった。デバッグ（バグ直し）も全部お願いした。

💰 料金メモ
claude.ai は無料プランあり（メッセージ数に上限あり）。Claude Code は有料プラン（Pro）が実質必要。月$20〜。
🌐
DeepL API
翻訳サービス — DeepL
▼

高精度な翻訳エンジン。APIとして使うと、自分のアプリに翻訳機能を組み込める。「日本語→英語・フランス語・韓国語…」をボタン一発でやってくれる。

このアプリでの役割：フレーズ入力時の自動翻訳。日本語を入力して「翻訳」ボタンを押すと各言語に変換される。

✓ 無料の範囲
月500,000文字まで無料（DeepL API Free）。個人の語学学習なら余裕でおさまる量。クレジットカード登録不要。
🐙
GitHub
コード管理 — Microsoft
▼

コードを保管・管理する場所。Googleドキュメントのコード版みたいなイメージ。変更履歴が全部残るので「前のバージョンに戻したい」が簡単にできる。Vercelと連携して公開するためにも必要。

このアプリでの役割：language-note.htmlの保管場所。更新するたびにここにアップロードする。

✓ 無料の範囲
個人の公開・非公開リポジトリは無料で無制限。今回の用途は完全無料でOK。
▲
Vercel（またはNetlify）
ホスティング / 公開
▼

GitHubに置いたファイルをURLとして公開してくれるサービス。「https://〇〇.vercel.app」みたいなアドレスが発行される。GitHubを更新するだけで自動で反映される。

このアプリでの役割：スマホや別端末からアクセスできるようにする。手元のHTMLファイルをウェブ公開する。

✓ 無料の範囲
個人プロジェクトは無料プランで十分。アクセス数が少ないうちは完全無料。商用・大規模運用になったら有料プランへ。
▲
Google ドライブ
ファイル保管 — Google
▼

おなじみのクラウドストレージ。アプリのデータ（フレーズ一覧）をJSONファイルとしてエクスポートして、ここに保管しておく。

このアプリでの役割：データのバックアップ。端末を変えたとき・アプリをリセットしたときに読み込み直せる。

✓ 無料の範囲
15GBまで無料。JSONファイルは数KBなので容量はほぼ気にしなくていい。

ブラウザ機能
Web Speech API（音声読み上げ）
🔊 ボタンを押すとフレーズを読み上げてくれる機能。追加料金なし、ブラウザが持ってる機能をそのまま使ってる。
ブラウザ機能
localStorage（データ保存）
入力したフレーズをブラウザの中に保存する仕組み。ページを閉じても消えない。スマホのメモ帳が端末に保存されるのと同じ感覚。
ブラウザ機能
File System Access API（ファイル保存）
Chromeなどで「保存先を選んでダウンロード」できる機能。Safariでは使えないので、自動的に通常のダウンロードに切り替わるようになってる。
外部フォント
Google Fonts（Noto Sans JP / DM Mono）
日本語と英語を混在させるためのフォント。無料。インターネット接続時に読み込まれる。
