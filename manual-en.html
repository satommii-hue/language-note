01 — B

Will it keep working?

The app runs on four services.

▲
Vercel
Hosts the app. A free hosting service used by developers worldwide
🐙
GitHub
Stores the code. Operated by Microsoft
🌐
DeepL
Powers auto-translation. The only service here with a paid tier — but the free plan is more than enough
☁️
Cloud storage
For backups. Use Google Drive, iCloud, Dropbox — whatever you prefer
Current assessment

The chances of this going away are close to zero. Vercel and GitHub are industry standards — they're not going anywhere suddenly.

Browser compatibility

The app runs on plain HTML and JavaScript with no exotic dependencies, so it's designed to work on older browsers too. The one exception: audio playback uses the browser's Speech Synthesis API, so on very old browsers you might see "audio not supported." Everything else — adding phrases, editing, tags, export — works fine even on a five-year-old phone.

Risk of data loss

The app itself won't delete your data, but it can disappear if you clear your browser cache, switch devices, or reinstall the app. Regular exports are recommended.

03

Setting up translation

Auto-translation requires a free DeepL account. The free tier covers 500,000 characters per month — you're very unlikely to hit that limit with everyday use.

01
Create a DeepL account
Go to deepl.com/en/pro-api and click "Get API key for free." Register with your email address. A credit card is required, but you won't be charged within the free 500K character limit — and everyday use won't come close to that.
02
Copy your API key
After logging in, go to "Account" → "API Keys." Copy the key shown (a long string of letters and numbers).
03
Enter it in the app
Open the app → tap "SET" in the bottom tab bar → "DeepL API Key" → "Set key" → paste the key and save.
NOTE

Your API key is stored only in your browser on this device. No one else can see it. You'll need to re-enter it if you switch devices.

04

Backup & syncing data

Phrase data is stored in your browser. To use the same data on multiple devices, or to back it up, use export/import.

export
Export → save to Google Drive
Tap "SET" in the bottom tab → "Export JSON file" → choose Google Drive as the save location (Chrome will open a save dialog). In Safari, the file saves to Downloads — move it to Google Drive from there.
import
Google Drive → Import
Download the JSON file from Google Drive → in the app, tap "Select JSON file" and choose it. Data will be added to your existing phrases — duplicates are skipped automatically.
NOTE

Any file storage works — Google Drive, iCloud, Dropbox, OneDrive, or wherever you prefer.

Google Drive free storage 15 GB
Each phrase is roughly 1KB. Even 1,000 phrases takes less than 1MB. Storage is basically a non-issue.
05

How to use it

add
Add a phrase
Go to the "ADD" tab → type in any language you like → tap "✨ Auto-translate" → review the results → tap "Add." You can attach tags to filter by category later.
NEPALI

Nepali (NE) is not supported by DeepL, so auto-translation won't work for that language. Translations need to be entered manually.

edit
Edit or delete a phrase
In the "LIST" tab, long-press a card (0.5 sec) → an edit modal opens where you can change content or delete the phrase.
view
See translations · hear audio
Tap a card to expand its translations. Use the ▶ button next to each language to play audio.
settings
Manage languages
Tap "SET" in the bottom tab → "Manage languages" to show or hide languages. Hiding a language doesn't delete its data.
archive
Archive a phrase
Long-press a card → tap "ARCHIVE" in the edit modal → the phrase moves to the ARCHIVE tab and disappears from the main list. It's not deleted, so you can restore it anytime.
06

Using it offline

The app isn't fully offline-capable, but there's a lot you can still do without a connection — in airplane mode or in the middle of nowhere.

✅
Browsing phrases
All saved phrases are accessible. Data lives on your device — no connection needed
✅
Audio playback
Uses the browser's built-in speech engine — works offline
✅
Editing & deleting
Editing existing phrases works offline too
❌
Auto-translation (DeepL)
Translation requires an internet connection and won't work offline
⚠️
Loading the app itself
It might load from browser cache, but that's not guaranteed. If you know you'll need it offline — before a trip, for example — open it online at least once beforehand
Tip for travel

Add and translate your phrases at home over Wi-Fi. Once they're saved, you can browse and play them anywhere — on the plane, no signal, wherever.
